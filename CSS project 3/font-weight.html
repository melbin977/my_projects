Font-Weight

This property specifies the weight or boldness of the font. Possible values for this property are 100-900, bold, border, lighter and normal. The numeric values 100-900 specify font weights where each value is a litter darker than its prwdecesssor. The value 400 is taken as Normal font weight,whereas 700 is Bold font weight.

Syntax: Font-weight:value;

where value=Lighter|Normal|Bold|Bolder|100|200|300|400|500|600|700|800|900
