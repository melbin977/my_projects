Font-Size

The Font-size property is used to set the size of a font. Its value can be absolute, relative size, length, or percentage.

An absolute value is fixed. It sets the font size to a definite value. it is specified by using one of the following keywords:

  • xx-small
  • x-small
  • small
  • large
  • x-large
  • xx-large

    • The most common units for specifying font-size are px(Pixels),pt(Point-size),%(Percentage), em, etc.The units px and pt make the font absolute. The em unit is defined relatively to the current font size . For example, 0.5 em is half the current font size. By Default, the font size is 16 pixels.
