Font-Style

The Font-Style property allows the text to be displayed in one of the three ways: normal,italic,oblique within a font-family. Italic forms are generally cursive whereas oblique displays sloped version of the regular font. If no font labelled as 'italic' is available, the 'oblique' version is picked up by the browser and vice versa.

  • Font-style:value;
    where value= normal|italic|oblique

  • H1{Font-style:oblique}
