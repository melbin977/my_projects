Font-Family

Font-Family specifies the preferred choice for the font of a particular element on a web page.It contains a list of font family names followed by generic family, all seprated by commas. The generic family name is optional, but it is advisable to add it as the last value in the list.If the browser does not have the specified font name,it will pick a similar font from the generic font-family.This propeerty accepts two values:

  • Family-name:The name of a font is its family-name,like "Georgia","Arial","Helvetica",etc.

  • Generic-family:Generic families are groups of family names that have uniform appearances.For example, Cursive,Monospace,Sans-Serif,etc.

  • Example: P{Font-Family:Arial,Verdana,Sans-serif;}
